# Functionalities WordPress Theme

A modern, WordPress Dashboard-inspired theme completely matching the Functionalities website UI.

## Features

- **Dashboard Design**: Premium, minimal UI inspired by the WordPress admin area.
- **Full Customizer Support**:
  - Live preview colors (Primary, Success, Warning).
  - Custom Hero section with badge, title, description, and stats.
  - Header & Footer options (Top bar toggle, copyright, social links).
  - Multi-layout blog settings.
- **Built-in Interactions**:
  - Native FAQ accordion logic.
  - Copy-to-clipboard functionality for snippets.
  - Responsive mobile menu.
- **Page Templates**:
  - `FAQ Page`: For searchable, accordion-style questions.
  - `Modules Page`: Grid layout for features/modules.
  - `Documentation Page`: Documentation layout with a sticky sidebar.

## Installation

1. Upload the `functionalities-theme` folder to your `/wp-content/themes/` directory.
2. Activate the theme via **Appearance > Themes**.
3. Go to **Appearance > Customize** to set up your primary colors and hero section.

## Recommended Menu Structure

- **Primary Menu**: Used for main navigation in the header.
- **Top Bar Menu**: Used for links in the dark admin bar at the top.
- **Footer Menus**: Three separate areas for footer columns.
